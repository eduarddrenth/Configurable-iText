2.Introduction

VectorPrint develops useful, robust java software libraries. At the moment there are five libraries. A general purpose library, a library that provides configuration functionality,  a powerfull library for developing reports based on iText, an SVG and a jFreeChart support library for reports.

Getting started.

----------------

  1 create a properties file with stylinginformation, example in junit test

  2 annotate your dataclasses with the available annotations Element and DataType, examples in junit tests

  3 extend com.vectorprint.report.data.DataCollectorImpl, example in junit test

  4 setup your classpath to include your classes, VectorPrintReport*.jar and dependent jars (see lib folder in binary distribution)

  5 java -cp <cp> com.vectorprint.report.running.ReportRunner <properties file> -dataclass <data class from step 3>

  You can look at the junit tests to see a working example

  Javadoc is recommended as a source for further details
